I'm about to wipe my computer again, so I've decided to make a list of software I use every day.  Everything is free and/or Free except for two...

Google Pack - easy to install as a first step, includes:

  • Mozilla Firefox with Google Toolbar - the most important thing you can install; just say no to internet explorer
  • Picasa - Photo organizer/editor
  • Google Earth - 3D Earth browser, the ultimate catrography software
  • Google Desktop - Desk sidebar
  • Google Talk - Voice and IM application
  • + Trillian

Communication

  • Gaim - multi-protocol chat program: aim, yahoo, microsoft ... gave trillian another try because it's in Google pack, and it's snappier
  • Skype - chat, voice, and pc <--> landline
  • (so I still have three IM applications running, oh well)
  • Azureus - bittorrent
  • + 7-zip - .zip .rar .tar .gz .7z

Office / Web Dev

  • OpenOffice - only really need it when somebody sends me a .doc or I need to make a pdf
  • Gimp - does everything photoshop does
  • Putty - shell client
  • FileZilla - ftp
  • Macromedia Studio (not free) - mostly Flash & Dreamweaver

Playerz

  • IrfanView - shows all things still + image cropping and converting
  • FooBar2000 - plays all things audio
  • VLC media player - plays all things video
  • Quicktime Pro (not free) - useful for doing simple video edits and conversions

Did I miss anything? Does anything get the job done better?
