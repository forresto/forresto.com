So I set out to solve whatever mistake I had made before, but ended up turning it into something else neat. Now it really is more like a spirograph. I guess the eventual goal with this will be to be able to switch between styles and input a number to which to skip. This version turns into a ring after awhile, so it is less interesting than the other after around 500. It might open up again though, but I didn't let it go that long. I didn't change the code much at all: 

// reset spiral movie
this.firstMandala.clear();
this.firstMandala.moveTo (250, 0);
// draw lines
with (this.firstMandala) {
	lineStyle(0, 0x000000, 50);
	for (i=0; i != points_count; i++) {
		next_x = radius*Math.cos(360/points_count*i);
		next_y = radius*Math.sin(360/points_count*i);
		lineTo(next_x, next_y);
	}
}
