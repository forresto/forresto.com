What a neat mistake!
This isn't at all what I wanted it to do, but it's a whole lot cooler. I don't know how long it would take for this tiny flash app to just be drawing a solid black circle and using cpu, but in the mean time it makes some pretty neat images. Click the number to pause, then doubleclick to step one forward. Click the center to reset. Here is the Actionscript code that is somehow beautifully wrong: 

// reset spiral movie
this.firstMandala.clear();
// draw lines
with (this.firstMandala) {
	lineStyle(0, 0x000000, 30);
	for (i=0; i != points_count; i++) {
		next_x = radius*Math.cos(360/points_count*i);
		next_y = radius*Math.sin(360/points_count*i);
		moveTo(0, 0);
		lineTo(next_x, next_y);
	}
}
