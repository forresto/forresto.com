Changing the lineTo() to a curveTo() gives these results, which are more fun in the big numbers than the last. Also, if you press F11 your browser might go full-screen for you. 

// reset spiral movie
this.firstMandala.clear();
this.firstMandala.moveTo (250, 0);
// draw lines
with (this.firstMandala) {
	for (i=1; i != points_count; i++) {
		lineStyle(0, 0x000000, 50);
		next_x = radius*Math.cos(360/points_count*i);
		next_y = radius*Math.sin(360/points_count*i);
		curveTo(0, 0, next_x, next_y);
	}
}
